## Required

that is required to be installed
- React-native-cli
- NPM (Node Package Manager)
- Emulator

## Clone or download
- `$ git clone https://github.com/auliarampit/JalaTest.git`
- `$ cd JalaTest`
- `$ npm install`

## Start Application
  `$ react-native run-android`
